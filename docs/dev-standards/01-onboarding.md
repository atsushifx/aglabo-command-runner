# Onboarding Guide

New contributor setup guide for this project.

## Prerequisites

- **GitHub account** with SSH key or Personal Access Token
- **Git** installed locally
- **Node.js** >= 20
- **pnpm** >= 10.24.0
- **Windows** (primary platform, WSL compatible)

## Initial Setup

### 1. Fork and Clone

```bash
# Fork on GitHub first
# Then clone your fork
git clone git@github.com:[YOUR_USERNAME]/aglabo-command-runner.git
cd aglabo-command-runner

# Add upstream remote
git remote add upstream git@github.com:atsushifx/aglabo-command-runner.git
```

### 2. Install Dependencies

```bash
# Install Node.js dependencies
pnpm install

# Install Git hooks
pnpm prepare
```

### 3. Install Development Tools (Windows)

```powershell
# Automated installation
.\scripts\install-dev-tools.ps1
.\scripts\install-doc-tools.ps1
```

**Installs**:

- Via Scoop: lefthook, delta, gitleaks
- Via Scoop (continued): dprint
- Via WinGet: dotenvx
- Via pnpm: commitlint, secretlint, cspell
- Via pnpm (continued): textlint, markdownlint

## Verify Setup

```bash
# Check installed tools
node --version
pnpm --version
git --version
lefthook version

# Run formatters/linters
pnpm format:dprint
pnpm lint:filenames
pnpm check:spells
pnpm lint:secrets
```

## First Contribution

### 1. Create Branch

```bash
# Create feature branch
git checkout -b feature/your-feature-name
```

### 2. Make Changes

Edit files following project standards (see `docs/dev-standards/`).

### 3. Quality Check

```bash
# Format code
pnpm format:dprint

# Lint
pnpm lint:filenames
pnpm lint:text
pnpm lint:markdown

# Spell check
pnpm check:spells

# Security scan (CRITICAL)
pnpm lint:secrets
```

### 4. Commit

```bash
# Stage changes
git add .

# Commit (message auto-generated)
git commit

# Push
git push origin feature/your-feature-name
```

**Note**: Commit messages are automatically generated by AI. Review and edit if needed.

### 5. Create Pull Request

Create PR on GitHub. The automated commit message will help with the PR description.
